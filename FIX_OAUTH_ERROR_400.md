# 🚀 Как Исправить Ошибку "Failed to create record"

## ❌ Текущая проблема

Вы видите в логах:
```
[PBAuthHelper] ✗ Failed to create user: ClientResponseError 400: Failed to create record.
```

Это значит что **PocketBase Admin API rules запретили создание новых пользователей**.

---

## ✅ Решение (5 шагов)

### Шаг 1️⃣: Откройте PocketBase Admin
```
http://pocketbase-n133ucx2k7b95oxc9qw0qtuw.176.112.158.15.sslip.io/_/
```

### Шаг 2️⃣: Перейдите в Collections → users
```
Left sidebar → Collections → выберите "users"
```

### Шаг 3️⃣: Нажмите на иконку "🛡️ API Rules"
```
Рядом с названием "users" вверху должна быть кнопка с иконкой щита 🛡️
Нажмите её
```

### Шаг 4️⃣: Установите Create rule
В поле **"Create"** введите:
```
@request.auth.id = ""
```

Это означает: **"Любой пользователь (даже неавторизованный) может создавать новые записи"**

### Шаг 5️⃣: Нажмите "Save"
```
Кнопка обычно внизу справа
```

---

## 📸 Какой результат должен быть

После сохранения, в API Rules должно быть:

| Правило | Значение |
|---------|----------|
| **Create** | `@request.auth.id = ""` ✅ |
| **Read** | (любое значение) |
| **Update** | (любое значение) |

---

## 🔄 Что дальше?

1. **Закройте браузер** (все вкладки с приложением)
2. **Очистите cookies**: DevTools → Application → Cookies → Delete All
3. **Перезагрузите страницу**: http://localhost:3000
4. **Попробуйте логин с Google** снова
5. **Проверьте server логи** - должны быть сообщения без ❌ ошибок

---

## 🐛 Если после этого ошибок нет

Тогда ищите в server логах:
```
[PBAuthHelper] ✓ User created successfully
[JWT Callback] ✓ Got PB token
```

Если видите ✓ - значит OAuth работает 🎉

---

## ❓ Если вы не видите API Rules кнопку

Попробуйте:
1. **Обновите страницу админки** (F5)
2. **Проверьте что выбрана коллекция "users"** (а не другая)
3. **Проверьте права доступа** - нужны админские права

---

## 📞 Если ничего не помогает

Поделитесь:
1. **Скриншот из Admin → Collections → users** (где API Rules)
2. **Полный текст ошибки их server логов**
3. **Какая версия PocketBase у вас в облаке?**
